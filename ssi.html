USABILITY EVALUATION
OF THE SSI DIGITAL LEARNING WEB APPLICATION

MSc UX DESIGN Project
Role

UX Researcher & Designer

The SSI Digital learning is a web application, i.e. an online learning portal that offers SSI diving training courses. The purpose of this evaluative study was to understand the issues first time users of the web application run into while exploring the digital learning.

The aim was to examine the existing web application and understand how users interact with the interface and perform a usability assessment of the current use of the SSI’s digital learning application in order to identify usability issues.

The project also focused on the behaviour of the application and aspects of the interface layout by observing users while they performed tasks in real time. Therefore, usability testing was conducted with 5 participants where they used the think-aloud protocol to vocalize their thoughts while going through given tasks.

The report provided findings of the testing and recommendations for usability implementation, and severity of identified issues.

Deliverables: Usability & Design Report, User Testing Script.

Methods employed: User research, heuristic evaluation, think aloud protocol, wireframes, high fidelity prototype.
